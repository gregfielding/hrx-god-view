# Addendum 1: Recruiter UI for Category Scores

## Objective
Make category scores visible, actionable, and intuitive for recruiters.

---

## 1. Where scores appear

### A. Applicant Table (Job Order)
Add columns:
- Reliability
- Punctuality
- Work Ethic
- Team Fit
- Job Readiness
- Stability

Display:
- 0–100 score
- color band (green/yellow/red)

---

### B. Applicant Detail View
Add a “Category Scores” panel:
- radar chart or horizontal bars
- show score + confidence
- show “Top Strengths” + “Concerns”

---

### C. Worker Profile (Global)
Persistent “Worker Reputation” section:
- current category scores
- last updated
- recent contributing events

---

## 2. Filters / Search

Allow filtering by:
- minimum reliability
- minimum punctuality
- minimum team fit
- gig-ready (derived from jobReadiness + preferences)

---

## 3. Recruiter Actions

Enable:
- sort by any category
- quick badges: “High Reliability”, “Low Punctuality”
- highlight mismatches (e.g. strong worker but poor team fit)

---

## 4. UI Principles

- Simple first (numbers + color)
- Drill-down optional (evidence)
- No overwhelming detail by default

---

## 5. Future

- “Top 10 workers for this job” based on category weighting
- client-specific weighting
