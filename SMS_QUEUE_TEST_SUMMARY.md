## 🧪 Test Execution Options

### Option 1: Via Firebase Console (Manual)

1. **Deploy functions first:**
   ```bash
   cd functions
   npm run deploy
   ```

2. **Create test request in Firestore Console:**
   - Go to: https://console.firebase.google.com/project/hrx1-d3beb/firestore
   - Navigate to: `tenants/{tenantId}/smsOutboundRequests`
   - Click "Add document"
   - Use auto-generated ID
   - Add fields:
     ```json
     {
       "tenantId": "your-tenant-id",
       "toPhoneE164": "+1234567890",
       "body": "Test message",
       "source": "manual",
       "status": "queued",
       "attemptCount": 0,
       "createdAt": [server timestamp]
     }
     ```
   - **Note:** `idempotencyKey` will be auto-generated by `createOutboundRequest()` function
   - For manual creation, you'd need to calculate it or use the callable function

### Option 2: Via Callable Functions (Recommended)

**From frontend or Firebase CLI:**

```javascript
// Test: Create request
const testCreate = httpsCallable(functions, 'testCreateOutboundRequest');
const result = await testCreate({
  tenantId: 'your-tenant-id',
  toPhoneE164: '+1234567890',
  body: 'Test message',
  threadId: 'optional-thread-id',
});

console.log('Request ID:', result.data.requestId);

// Test: Check status
const testCheck = httpsCallable(functions, 'testCheckRequestStatus');
const status = await testCheck({
  tenantId: 'your-tenant-id',
  requestId: result.data.requestId,
});

console.log('Status:', status.data.request.status);

// Test: Idempotency
const testIdemp = httpsCallable(functions, 'testIdempotency');
const idempResult = await testIdemp({
  tenantId: 'your-tenant-id',
  toPhoneE164: '+1234567890',
  body: 'Same message twice',
  threadId: 'optional-thread-id',
});

console.log('Idempotent?', idempResult.data.isIdempotent);
```

### Option 3: Via Firebase CLI

```bash
# Create test request
firebase functions:call testCreateOutboundRequest --data '{
  "tenantId": "your-tenant-id",
  "toPhoneE164": "+1234567890",
  "body": "Test message"
}'
```
